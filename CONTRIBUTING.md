# DaiC-Plugins Contributor Guide

## Introduction

### What is DaiC?

DaiC (Decompiler AI Collaborative) is a collaborative software for reverse engineering. It will use NLP to decompile assembly language into pseudo code to help the user. DaiC answers accessibility questions that most other competitors face.

The development of DaiC is still in its early stages, so please bare with us as we build out this guide.

### What is this guide?

This guide is for anyone who wants to contribute to DaiC using our plugin system. It's a work in progress, and will be updated regularly.

You can use our [plugin documentation](https://daic.re/docs/) and read existing plugins for a first approach.

### How can I contribute?

There are many ways to contribute to DaiC-Plugins. You can:

- [Report a bug](https://github.com/DaiC-re/DaiC-Plugins/issues/new)
- Submit a PR
